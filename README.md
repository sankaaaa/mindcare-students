# MindCare

**MindCare** це вебплатформа для психологічної підтримки військових, ветеранів та їхніх родин.  
Система дає змогу пацієнтам знаходити фахівців, переглядати їхні профілі, бронювати консультації,
а спеціалістам відповідно керувати власним профілем, записами та доступними часовими слотами.

Проєкт створено на базі **React**, з використанням **Supabase** як бази даних і **Node.js + Express + Nodemailer**
для надсилання email-сповіщень.

---

## Основні можливості

### Для пацієнтів
- реєстрація та вхід в обліковий запис;
- перегляд списку фахівців;
- перегляд детальної інформації про спеціаліста;
- бронювання консультацій через календар;
- перегляд власних записів;
- скасування записів;
- редагування особистих даних;
- відновлення пароля;
- отримання email-сповіщень про реєстрацію, запис, скасування запису та зміну пароля.

### Для спеціалістів
- реєстрація та вхід в обліковий запис;
- редагування інформації профілю;
- перегляд власних записів;
- додавання доступних слотів у календар;
- видалення вільних слотів;
- скасування записів пацієнтів;
- отримання email-сповіщень про нові записи та скасування консультацій.

## Підготовка до запуску

Перед запуском програми встановіть усі залежності:

### `npm install`

Після цього перейдіть до папки конфігурації в папці src:

### `cd src/config`

Далі вам необхідно запустити окремо сервер для коректної роботи поштової частини:

### `node server.js`

---

## Запуск програми

Після запуску сервера перейдіть до кореневої папки та запустіть клієнтську частину:

### `npm start`
